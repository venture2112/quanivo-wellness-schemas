# Quanivo Wellness - AEO Schema Repository

This repository contains JSON-LD schema markup files for Quanivo Wellness website to optimize for AI search engines (ChatGPT, Perplexity, Google AI Overviews, Claude).

## Schema Files

### Page Schemas
- `quanivo-faq-schema.json` - FAQPage schema with 10 Q&As
- `quanivo-organization-schema.json` - Organization + parent company (Venture Works Inc.)
- `quanivo-localbusiness-schema.json` - LocalBusiness with services
- `quanivo-blog-schema.json` - Blog + Article schema for all posts
- `quanivo-breadcrumb-schema.json` - BreadcrumbList navigation

### Product Schemas (7 Products)
1. `quanivo-product-pod-schema.json` - Bio-Wellness Pod
2. `quanivo-product-recliner-schema.json` - Bio-Wellness Energy Recliner
3. `quanivo-product-cryobike-schema.json` - Cryo Bike
4. `quanivo-product-cryotreadmill-schema.json` - Cryo Treadmill
5. `quanivo-product-hyperbaric-schema.json` - Hyperbaric Chamber
6. `quanivo-product-massagechair-schema.json` - Zero Gravity Massage Chair
7. `quanivo-product-redlight-schema.json` - Red Light Therapy Panels

## How to Use

1. Download the schema file you need
2. Copy the JSON-LD code
3. Paste into your GHL page's Custom Code section (in the `<head>`)
4. Save and publish

## Benefits

- ✅ Makes content visible to AI search engines
- ✅ Enables rich snippets in Google
- ✅ Increases chances of being cited by ChatGPT, Perplexity, Claude
